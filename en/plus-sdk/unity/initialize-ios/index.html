ЩҶШөШЁ Cocoapods

ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҮШұЪҶЩҮ ШіШ§ШҜЩҮ ШӘШұ ШЁШұЩҲЫҢ Cocoapods Щ…ШӘШҙШұ ШҙШҜЩҮ Ш§ШіШӘ. ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ ШӘШ§ ШЁЩҮ ШӯШ§Щ„ Cocoapods ШұШ§ ШұЩҲЫҢ ШіЫҢШіШӘЩ… ШӘШ§ЩҶ ЩҶШөШЁ ЩҶЪ©ШұШҜЩҮ Ш§ЫҢШҜШҢ Щ…ЫҢВӯШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ ШҜШіШӘЩҲШұ ШІЫҢШұ Ш§ЫҢЩҶ Ъ©Ш§Шұ ШұШ§ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЫҢШҜ:

$ sudo gem install cocoapods

ШЁШұШ§ЫҢ Ъ©ШіШЁ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШЁЫҢШҙШӘШұ Щ…ЫҢВӯШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ ЩҲШЁШіШ§ЫҢШӘ Cocoapods Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ

Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШЁЩҮ Podfile

Ш®Ш·ЩҲШ· ШІЫҢШұ ШұШ§ ШЁЩҮ Podfile Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ:

pod вҖҳTapsellPlusSDKвҖҷ, вҖҳ1.0.0вҖҷ

ШЁШ§ Ш§ЫҢЩҶ Ш®Ш·ШҢ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ЩҲ ШӘЩ…Ш§Щ…ЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ШўЩҶ ШҜШ§ЩҶЩ„ЩҲШҜ ШҙШҜЩҮШҢ ШӘЩҶШёЫҢЩ…Ш§ШӘ Щ„Ш§ШІЩ… ШЁШұШ§ЫҢ ЩҮШұ Ъ©ШҜШ§Щ… Ш§Ш№Щ…Ш§Щ„ ШҙШҜЩҮ ЩҲ ШЁЩҮ ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ Ш§Ш¶Ш§ЩҒЩҮ Щ…ЫҢВӯЪҜШұШҜЩҶШҜ.

ШўЩҫШҜЫҢШӘ ЩҒШ§ЫҢЩ„ .plist ЩҫШұЩҲЪҳЩҮ

ШЁШ§ Ш§ЩҶШӘШҙШ§Шұ iOS 9 , Ш§ЩҫЩ„ ШіЫҢШіШӘЩ… ATS ШұШ§ Щ…Ш№ШұЩҒЫҢ ЩҶЩ…ЩҲШҜ Ъ©ЩҮ Ш·ШЁЩӮ ШўЩҶ Щ„Ш§ШІЩ… Ш§ШіШӘ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ Ш§ШІ SSL ШЁШ§ Щ…ШҙШ®ШөШ§ШӘ Ш®Ш§ШөЫҢ ШЁШұШ§ЫҢ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШҙШЁЪ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЩҶШҜ. ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶ШұШҢ Щ„Ш§ШІЩ… Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘШЁЩ„ЫҢШәШ§ШӘ ШӘЩҫШіЩ„ ATS ШҜШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШәЫҢШұЩҒШ№Ш§Щ„ ШҙЩҲШҜ. Ш¬ЩҮШӘ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ Щ…ШҜШ§Ш®Щ„ЩҮ ATS ШҜШұ ЩҫШ®Шҙ ШӘШЁЩ„ЫҢШәШ§ШӘШҢ ЩӮШіЩ…ШӘ ШІЫҢШұ ШұШ§ ШЁЩҮ plist ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.

<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>

Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ URL Schemes

ШЁШ§ Ш§ЩҶШӘШҙШ§Шұ ЩҶШіШ®ЩҮ iOS 9ШҢ ШҙШұЪ©ШӘ Ш§ЩҫЩ„ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШЁЫҢЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ (canOpenUrl) ШұШ§ Щ…ШӯШҜЩҲШҜ Ъ©ШұШҜЩҮ Ш§ШіШӘ ЩҲ ЩҮШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШЁШ§ЫҢШҜ Щ…ШҙШ®Шө Ъ©ЩҶШҜ ЪҶЩҮ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұЫҢ ШұШ§ ШЁШ§ШІ Ш®ЩҲШ§ЩҮШҜ ЩҶЩ…ЩҲШҜ. ШӘЩҫШіЩ„ Ш§ШІ ШӘШ§ШЁШ№ canOpenUrl ШЁШұШ§ЫҢ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢ ШҜШұ Щ…ЩҲШұШҜ Ш§ЩҶШ¬Ш§Щ… Ш№Щ…Щ„ ЩҶЩҮШ§ЫҢЫҢ ШҜШұ ШЁЩҶШұЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШұШ§ЫҢ ЩҒШ№Ш§Щ„ШіШ§ШІЫҢ deep-linking ШЁЫҢЩҶ SDK ШӘЩҫШіЩ„ ЩҲ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒШҢ ШЁШ®Шҙ ШІЫҢШұ ШұШ§ ШЁЩҮ plist ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.

<key>LSApplicationQueriesSchemes</key>
<array>
    <string>mailto</string>
    <string>itms-apps</string>
    <string>sms</string>
    <string>tel</string>
</array>

ШўЩҫШҜЫҢШӘ ШӘЩҶШёЫҢЩ…Ш§ШӘ Build

ЩҒЩ„ЪҜ вҖ“ObjC ШұШ§ ШЁЩҮ Other Linker Flags ШҜШұ Build Setting Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ. ШіЩҫШі ЩҒЩ„ЪҜ Always Embed Swift Standard Libraries ШұШ§ ШҜШұ ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ ЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜ.
