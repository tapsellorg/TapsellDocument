ШіШ§ЫҢШұ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…Ш·Ш§ШЁЩӮ Ш§ЫҢЩҶ ШўЩ…ЩҲШІШҙ ШЁЩҮ ЩҫШұЩҲЪҳЩҮ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶ ШЁШ§ ШұЩҲШҙвҖҢЩҮШ§ЫҢ ШӘШіШӘ ЪҜЩҒШӘЩҮ ШҙШҜЩҮ Щ…Ш·Щ…ШҰЩҶ ШҙЩҲЫҢШҜ ШЁЩҮ ШҜШұШіШӘЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

ШЁШұШ§ЫҢ Ъ©ШіШЁ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШЁЫҢШҙШӘШұ ШҜШұ Щ…ЩҲШұШҜ ЩҮШұ ad network Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ ЩҮЩ…Ъ©Ш§ШұШ§ЩҶ Щ…Ш§ ШҜШұ ШӘЫҢЩ… ШұШіШ§ЩҶЩҮ Ш§ШІ ЩӮШіЩ…ШӘ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШөШӯШЁШӘ Ъ©ЩҶЫҢШҜ.

ЩҶЪ©ШӘЩҮ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ AdMob

ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШҜЩҶШӘЩҲШұЪ© Google AdMob SDK (ЩҶШіШ®ЩҮвҖҢЫҢ 20.0.0 ШЁЩҮ ШЁШ№ШҜ) ШЁШ§ЫҢШіШӘЫҢ App ID ШўЩҶШұШ§ ШҜШұ Щ…Ш§ЩҶЫҢЩҒШіШӘ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ЩӮШұШ§Шұ ШҜЩҮЫҢШҜ.
Щ„Ш§ШІЩ… ШЁЩҮ Ш°Ъ©Шұ Ш§ШіШӘ ШҜШұ ШөЩҲШұШӘ Ш№ШҜЩ… ЩҲШ¬ЩҲШҜ Ш§ЫҢЩҶ ШӘЪҜ ШҜШұ Щ…Ш§ЩҶЫҢЩҒШіШӘ Ш®Ш·Ш§ЫҢ The Google Mobile Ads SDK was initialized incorrectly. ШұШ® Ш®ЩҲШ§ЩҮШҜ ШҜШ§ШҜ

ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ App ID Ш§ШЁШӘШҜШ§ Ш§ШІ ЫҢЪ©ЫҢ Ш§ШІ ШұШ§ЩҮвҖҢЩҮШ§ЫҢ ШІЫҢШұ Ш§ЫҢЩҶ ШҙЩҶШ§ШіЩҮ ШұШ§ Ш§ШІ ШӘЩҫШіЩ„ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ (ЩҫЫҢШҙ Ш§ШІ ШҜШұЫҢШ§ЩҒШӘ ШҙЩҶШ§ШіЩҮШҢ ШҙШұШ§ЫҢШ· ЩҒШ№Ш§Щ„вҖҢШіШ§ШІЫҢ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ Ш®Ш§ШұШ¬ЫҢ ШұШ§ ШҜШұ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© Щ…Ш·Ш§Щ„Ш№ЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ):

  • Ш§ШұШіШ§Щ„ Ш§ЫҢЩ…ЫҢЩ„ ШЁЩҮ publishers [at] tapsell.ir

  • ШӘЩ…Ш§Ші ШЁШ§ ШҙЩ…Ш§ШұЩҮ ШӘЩ„ЩҒЩҶ 5-88206893 (ШҜШ§Ш®Щ„ЫҢ Ыі - ЩҶШ§ШҙШұЫҢЩҶ)

ШіЩҫШі ШҙЩҶШ§ШіЩҮвҖҢЫҢ Щ…ЩҲШұШҜ ЩҶШёШұ ШұШ§ ШЁШ§ ЩҒШұЩ…ШӘ ШІЫҢШұ ШҜШұ ЩҒШ§ЫҢЩ„ android/app/src/main/AndroidManifest.xml ЩӮШұШ§Шұ ШҜЩҮЫҢШҜ:

ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮвҖҢЪ©ШұШҜЩҶ Щ…ЫҢЩҶШӘЪҜШұШ§Щ„ ЩҶЫҢШ§ШІ ШЁЩҮ VPN ЫҢШ§ ЩҫШұШ§Ъ©ШіЫҢ ШЁШ§ ЩҫШұЩҲШӘЪ©Щ„ Щ…ЩҶШ§ШіШЁ Щ…Ш§ЩҶЩҶШҜ Kerio ЫҢШ§ OpenVPN ШҜШ§ШұЫҢШҜ. ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ Ш§ЫҢЩҶ ШіШұЩҲШұЩҮШ§ ШӘЩҲШіШ· ШҙЪ©ЩҶ ЫҢШ§ FOD ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ

ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҙШЁЪ©ЩҮвҖҢЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ Unity Ads Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘ minSDK Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш®ЩҲШҜ ШұШ§ ЫұЫ№ ЩӮШұШ§Шұ ШҜЩҮЫҢШҜ. ЩҲ ЫҢШ§ Ш§ЫҢЩҶ Ъ©ЩҮ Ш®Ш· ШІЫҢШұ ШұШ§ ШЁЩҮ ЩҒШ§ЫҢЩ„ AndroidManifest.xml ЩҫШұЩҲЪҳЩҮвҖҢШӘШ§ЩҶ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

Ш§ЫҢЩҶ Ш®Ш· ШЁЩҮ Щ…ЩҶШёЩҲШұ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ЩҲЩӮЩҲШ№ Ш®Ш·Ш§ ШЁШұШ§ЫҢ ШЁЫҢЩ„ШҜ ШҙШҜЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ШЁШ§ minSDK Ъ©Щ…вҖҢШӘШұ Ш§ШІ ЫұЫ№ Щ…ЫҢвҖҢШЁШ§ШҙШҜ. Ш§ШІ ШўЩҶвҖҢШ¬Ш§ Ъ©ЩҮ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШЁЩҮ ШҙЪ©Щ„ ЩҮЩҲШҙЩ…ЩҶШҜ Щ…Ш§ЩҶШ№ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә Unity Ads ШЁЩҮ Ш§ЩҶШҜШұЩҲЫҢШҜЩҮШ§ЫҢ ШЁШ§ API Level Ъ©Щ…вҖҢШӘШұ Ш§ШІ ЫұЫ№ Щ…ЫҢвҖҢШҙЩҲШҜШҢ Щ…ШҙЪ©Щ„ЫҢ ШЁШұШ§ЫҢ ШўЩҶ ШҜШіШӘЩҮ Ш§ШІ Ъ©Ш§ШұШЁШұШ§ЩҶШӘШ§ЩҶ Ъ©ЩҮ ЩҶШіШ®ЩҮвҖҢЫҢ Ш§ЩҶШҜШұЩҲЫҢШҜ ЩӮШҜЫҢЩ…ЫҢ ШҜШ§ШұЩҶШҜ ЩҫЫҢШҙ ЩҶЩ…ЫҢвҖҢШўЫҢШҜ.
