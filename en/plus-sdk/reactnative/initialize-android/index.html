ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШҜШұ ReactNative (Android)

  • ШҜШұЫҢШ§ЩҒШӘ SDK ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші
  • ШҙШұЩҲШ№ Ъ©Ш§Шұ ШЁШ§ ЩҫЩ„Ш§ЪҜЫҢЩҶ

ШҜШұ ШөЩҲШұШӘ ЩҲШ¬ЩҲШҜ Щ…ШҙЪ©Щ„ ЫҢШ§ Ш§ШЁЩҮШ§Щ… ШЁЩҮ Щ„ЫҢШіШӘ Ш®Ш·Ш§ЩҮШ§ЫҢ ЩҒЩҶЫҢ ШұШ§ЫҢШ¬ Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ ЫҢШ§ ШөЩҒШӯЩҮвҖҢЫҢ Github Issues ШұШ§ ШЁШұШұШіЫҢ ЩҶЩ…Ш§ЫҢЫҢШҜ.

Ш®Ш·Ш§ЩҮШ§ЫҢ ШұШ§ЫҢШ¬ ЩҲ Ш§ШӯШӘЩ…Ш§Щ„ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұ GitHub issues ШұЫҢЩҫШ§ШІЫҢШӘЩҲШұЫҢ ЩҫЩ„Ш§ЪҜЫҢЩҶ Щ…ШҙШ§ЩҮШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШҜШұЫҢШ§ЩҒШӘ SDK ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші

ШЁШұШ§ЫҢ Ш§ЩҒШІЩҲШҜЩҶ ЩҫЩ„Ш§ЪҜЫҢЩҶ ШӘЩҫШіЩ„вҖҢЩҫЩ„Ш§Ші ШЁЩҮ ЩҫШұЩҲЪҳЩҮ Ш§ШІ Ъ©Ш§Щ…ЩҶШҜ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:

npm install react-native-tapsell-plus --save

ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШЁШ§ЫҢШіШӘЫҢ ЩҲШұЪҳЩҶ Android gradle plugin ШӯШҜШ§ЩӮЩ„ Ыі.Ы¶.Ы° ШЁШ§ШҙШҜ
ШЁШұШ§ЫҢ Щ…ШҙШ§ЩҮШҜЩҮвҖҢЫҢ ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ъ©ШҜ ЩҶШёЫҢШұ ШҜШұ ШіЩ…ЩҫЩ„ ШӘЩҫШіЩ„ ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© Щ…ШҙШ§ЩҮШҜЩҮ Ъ©ЩҶЫҢШҜ

// <prj_root>/android/build.gradle

dependencies { 
  classpath("com.android.tools.build:gradle:4.1.3") // Use a version higher than 3.5.x
}

ШҜШұ ШөЩҲШұШӘ ЩҫШ§ЫҢЫҢЩҶ ШЁЩҲШҜЩҶ ЩҲШұЪҳЩҶ ШЁШ§ Ш®Ш·Ш§ЩҮШ§ЫҢ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ЩҮЩ…Ш§ЩҶЩҶШҜ Manifest merger failed with multiple errors, see logs ШұЩҲШЁШұЩҲ Ш®ЩҲШ§ЩҮЫҢШҜ ШҙШҜ.

ШҙШұЩҲШ№ Ъ©Ш§Шұ ШЁШ§ ЩҫЩ„Ш§ЪҜЫҢЩҶ

Ш§ШЁШӘШҜШ§ Щ…Ш§ЪҳЩҲЩ„ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШұШ§ ШЁЩҮ Ъ©ШҜ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ:

import { TapsellPlus } from 'react-native-tapsell-plus';

ШіЩҫШі ШӘШ§ШЁШ№ ШІЫҢШұ ШұШ§ ШҜШұ ЫҢЪ©ЫҢ Ш§ШІ Ш§ШіЪ©ШұЫҢЩҫШӘвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Ш®ЩҲШҜ Ъ©ЩҮ ШҜШұ Ш§ШЁШӘШҜШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЫҢШҜ.

TapsellPlus.initialize(APP_KEY);

APP_KEY Ъ©Щ„ЫҢШҜ ШӘЩҫШіЩ„ ЩҮШіШӘ ЩҲ ШЁШұШ§ЫҢ ЩҮШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ъ©ЩҮ Щ…ЫҢвҖҢШіШ§ШІЫҢШҜ ШҜШұ ЩҫЩҶЩ„ ШӘЩҫШіЩ„ ШіШ§Ш®ШӘЩҮ Щ…ЫҢШҙЩҲШҜШҢ Щ…ЫҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ЩҫЩҶЩ„ Ъ©ЩҫЫҢ Ъ©ЩҶЫҢШҜ.

Ш§Ъ©ЩҶЩҲЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ ШӘЩҲШ¬ЩҮ ШЁЩҮ ЩҶЫҢШ§ШІ Ш®ЩҲШҜ ЩҲ ШӘЩҲШ¶ЫҢШӯШ§ШӘ ШЁЩҮ ЩҮШұ ЩҶЩҲШ№ ШӘШЁЩ„ЫҢШәШҢ ШӘШЁЩ„ЫҢШә Щ…ЩҲШұШҜ ЩҶШёШұ ШұШ§ ЩҶЩ…Ш§ЫҢШҙ ШҜЩҮЫҢШҜ.

ШЁШұШ§ЫҢ Щ…ШҙШ§ЩҮШҜЩҮвҖҢЫҢ Ш¬ШІШҰЫҢвҖҢШӘШұ Щ„Ш§ЪҜвҖҢЩҮШ§ ШҜШұ Щ„Ш§ЫҢШЁШұШұЫҢ Ш§ШІ Ъ©ШҜ

TapsellPlus.setDebugMode(3)

Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ЩҮЩҶЪҜШ§Щ… ШҜШұШ®ЩҲШ§ШіШӘ ЫҢШ§ ЩҶЩ…Ш§ЫҢШҙ ЩҮШұЪҜЩҲЩҶЩҮ Ш®Ш·Ш§ЫҢЫҢ ШЁШ§ Ш¬ШІШҰЫҢШ§ШӘ ШҜЩӮЫҢЩӮвҖҢШӘШұ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ targetSdkVersion ШЁШұШ§ШЁШұ ЫІЫё ЫҢШ§ ШЁШ§Щ„Ш§ШӘШұ ШЁШ§ШҙШҜ ЩҮЩҶЪҜШ§Щ… Ш§Ш¬ШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШҜШұ ШӯШ§Щ„ШӘ ШҜЫҢШЁШ§ЪҜ (ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Metro sever) ШЁШ§ЫҢШіШӘЫҢ network-security ШұШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮ Щ„ШӯШ§Шё Ъ©ЩҶЫҢШҜ.
ШЁШұШ§ЫҢ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШЁЫҢШҙШӘШұ Ш§ЫҢЩҶ ШЁШ®Шҙ Ш§ШІ Щ…ШіШӘЩҶШҜШ§ШӘ ШұЫҢвҖҢШ§Ъ©ШӘ ЩҶЫҢШӘЫҢЩҲ ШұШ§ Щ…Ш·Ш§Щ„Ш№ЩҮ Ъ©ЩҶЫҢШҜ.

ШұШ§ЩҮ ШӯЩ„ ШіШұЫҢШ№:
Ш№ШЁШ§ШұШӘ android:usesCleartextTraffic="true" ШұШ§ ШЁЩҮ Ш§ШӘШұЫҢШЁЫҢЩҲШӘЩҮШ§ЫҢ ШӘЪҜ application ШҜШұ Щ…Ш§ЩҶЫҢЩҒШіШӘ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ:

<!-- android/app/src/main/AndroidManifest.xml -->
<application
  ...
  android:usesCleartextTraffic="true">
 <!-- ... -->
</application>
