ШіЫҢШіШӘЩ… ШөШӯШӘ ШіЩҶШ¬ЫҢ

  • ЩҶШӯЩҲЩҮ Ъ©Ш§ШұЪ©ШұШҜ
  • ШҜШұЫҢШ§ЩҒШӘ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә

ЩҶШӯЩҲЩҮ Ъ©Ш§ШұЪ©ШұШҜ

ШіШұЩҲЫҢШі ШөШӯШӘ ШіЩҶШ¬ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ШӘЩҫШіЩ„ (S2S Verification) ШЁШұШ§ЫҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ ЩҲ ШЁШ§ШІЫҢвҖҢЩҮШ§ЫҢ ШўЩҶЩ„Ш§ЫҢЩҶ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ ШӘШ§ ШӘЩҲШіШ№ЩҮ ШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§ШІ ШөШӯШӘ ШӘШЁЩ„ЫҢШә ЩҶШҙШ§ЩҶвҖҢШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯШ§ШөЩ„ ЩҶЩ…Ш§ЫҢЩҶШҜ ЩҲ Ш§Щ…Ъ©Ш§ЩҶ ШіЩҲШЎ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ mod ШҙШҜЩҮ ШЁШ§ШІЫҢ ШәЫҢШұЩ…Щ…Ъ©ЩҶ ШҙЩҲШҜ.

ЩҶШӯЩҲЩҮ Ъ©Ш§ШұЪ©ШұШҜ ШіЫҢШіШӘЩ… ШөШӯШӘ ШіЩҶШ¬ЫҢ ШЁЩҮ Ш§ЫҢЩҶ ШөЩҲШұШӘ Ш§ШіШӘ Ъ©ЩҮ ЩҫШі Ш§ШІ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә ШҜШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶШҢ Щ…ЫҢШЁШ§ЫҢШіШӘ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә (suggestionId) ШЁШұШ§ЫҢ ШіШұЩҲШұ ШЁШ§ШІЫҢ ЫҢШ§ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш§ШұШіШ§Щ„ ШҙЩҲШҜШӣ ШіЩҫШі ШіШұЩҲШұ ШЁШ§ШІЫҢ/Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШЁШ§ Ш§ШұШіШ§Щ„ ЫҢЪ© ШҜШұШ®ЩҲШ§ШіШӘ ШЁЩҮ ШіШұЩҲШұ ШӘЩҫШіЩ„ШҢ Ш§ШІ Щ…Ш№ШӘШЁШұ ШЁЩҲШҜЩҶ ШҙЩҶШ§ШіЩҮ Ш§ШұШіШ§Щ„ ШҙШҜЩҮ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯШ§ШөЩ„ ЩҶЩ…Ш§ЫҢШҜ.

Ш¬ЩҮШӘ Ш§ШұШӘШЁШ§Ш· ШіШұЩҲШұ ШЁЩҮ ШіШұЩҲШұ Ш§ШІ ШўШҜШұШі ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

http://api.tapsell.ir/v2/suggestions/validate-suggestion

ШҜШұШ®ЩҲШ§ШіШӘ Ш§ШұШіШ§Щ„ ШҙШҜЩҮ ШЁШ§ЫҢШҜ ШЁШөЩҲШұШӘ POST ЩҲ Щ…ШӯШӘЩҲШ§ЫҢ Ш§ШұШіШ§Щ„ЫҢ ШЁШөЩҲШұШӘ json ШЁШ§ШҙШҜ. (Content-Type:application/json)

ЩҶШ§ШҙШұЫҢЩҶЫҢ Ъ©ЩҮ ЩӮШөШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШіЫҢШіШӘЩ… ШөШӯШӘ ШіЩҶШ¬ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ШӘЩҫШіЩ„ ШұШ§ ШҜШ§ШұЩҶШҜ Щ…ЫҢШЁШ§ЫҢШіШӘ ШҜШұШ®ЩҲШ§ШіШӘ Ш®ЩҲШҜ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ШӘЩ…Ш§Ші ШӘЩ„ЩҒЩҶЫҢ ЫҢШ§ Ш§ШұШіШ§Щ„ Ш§ЫҢЩ…ЫҢЩ„ ШЁЩҮ ШўШҜШұШі publishers@tapsell.ir ШЁЩҮ ЩҲШ§ШӯШҜ ЩҒЩҶЫҢ ШӘЩҫШіЩ„ Ш§Ш·Щ„Ш§Ш№ ШҜЩҮЩҶШҜ.

ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә ШЁШ§ЫҢШҜ ШҜШұ ШЁШҜЩҶЩҮ ШҜШұШ®ЩҲШ§ШіШӘ ЩҲ ШЁШөЩҲШұШӘ ШІЫҢШұ ЪҜЩҶШ¬Ш§ЩҶШҜЩҮ ШҙЩҲШҜ.

{
  "suggestionId":"SUGGESTION_ID"
}

ЩҫШ§ШіШ® ШЁШұЪҜШұШҜШ§ЩҶШҜЩҮ ШҙШҜЩҮ ШЁЩҮ ШҜШұШ®ЩҲШ§ШіШӘ Ш§ШұШіШ§Щ„ ШҙШҜЩҮ ЩҶЫҢШІ ШЁШөЩҲШұШӘ json ЩҲ Щ…Ш·Ш§ШЁЩӮ ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ ШІЫҢШұ Ш§ШіШӘ.

{
  "valid": true
}

ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ Щ…ЩӮШҜШ§Шұ ЩҫШ§ШұШ§Щ…ШӘШұ valid ШҜШұ ЩҫШ§ШіШ® ШЁШұЪҜШұШҜШ§ЩҶШҜЩҮ ШҙШҜЩҮ ШЁШұШ§ШЁШұ true ШЁШ§ШҙШҜШҢ ЫҢШ№ЩҶЫҢ ШҙЩҶШ§ШіЩҮ Ш§ШұШіШ§Щ„ ШҙШҜЩҮ Щ…Ш№ШӘШЁШұ Ш§ШіШӘ. ШҜШұ ШәЫҢШұ Ш§ЫҢЩҶШөЩҲШұШӘ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә Ш§ШұШіШ§Щ„ЫҢ Щ…Ш№ШӘШЁШұ ЩҶШЁЩҲШҜЩҮвҖҢШ§ШіШӘ.

ШҜЩӮШӘ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ЩҮШұ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә Ъ©ШҜЫҢ ЫҢЪ©ШЁШ§Шұ Щ…ШөШұЩҒ Ш§ШіШӘ ЩҲ Ш§ЪҜШұ ЫҢЪ©ШЁШ§Шұ ШўЩҶ ШҙЩҶШ§ШіЩҮ Щ…Ш№ШӘШЁШұ ШҙЩҶШ§Ш®ШӘЩҮ ШҙЩҲШҜШҢ ШҜШұ ШҜЩҒШ№Ш§ШӘ ШЁШ№ШҜЫҢ Щ…Ш№ШӘШЁШұ ШҙЩҶШ§Ш®ШӘЩҮ ЩҶШ®ЩҲШ§ЩҮШҜ ШҙШҜ.

ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ Postman ШұЩҲЫҢ ШҜЪ©Щ…ЩҮвҖҢЫҢ ШІЫҢШұ Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ.

ШҜШұЫҢШ§ЩҒШӘ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә

Щ…ЩӮШҜШ§Шұ SUGGESTION_ID ШұШ§ ШҜШұ Ъ©Ш§Щ„вҖҢШЁЪ© onAdAvailable ШЁЩҮ Ъ©Щ…Ъ© ШұЩҲШҙ ШІЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.

Tapsell.requestAd(CONTEXT,
        ZONE_ID,
        new TapsellAdRequestOptions(),
        new TapsellAdRequestListener() {
            @Override
            public void onAdAvailable(String adId) {
              suggustionId = adId;
            }

            @Override
            public void onError(String message) {
            }
        });

ЩҮЩ…ЪҶЩҶЫҢЩҶ ШЁШұШ§ЫҢ Ш§Ш·Щ„Ш§Ш№ Ш§ШІ ЩҲШ¶Ш№ЫҢШӘ ШҜШұЫҢШ§ЩҒШӘ Ш¬Ш§ЫҢШІЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ъ©Ш§Щ„вҖҢШЁЪ© onRewarded Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.
