вҡ  ЩҮШҙШҜШ§Шұ! Ш§ЫҢЩҶ SDK ШҜЫҢЪҜШұ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ вҡ 

Ш§ЪҜШұ ШӘШ§Ъ©ЩҶЩҲЩҶ ШЁШұШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҜШұЩҲЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш§ШІ ШӘЩҫШіЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ШұШҜЫҢШҜШҢ ШЁЩҮШӘШұ Ш§ШіШӘ ШІЫҢЩҶвҖҢЩҫШі Ш§ШІ ШӘЩҫШіЩ„вҖҢЩҫЩ„Ш§Ші Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ШӘЩҫШіЩ„ ЩҫЩ„Ш§ШіШҢ Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШҜШ§ШұШ§ ШЁЩҲШҜЩҶ ШӘЩ…Ш§Щ… Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ ШӘЩҫШіЩ„ШҢ Ш§Щ„ЪҜЩҲШұЫҢШӘЩ…вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙЩ…ЩҶШҜШ§ЩҶЩҮвҖҢШӘШұШҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Щ…ШӘЩҶЩҲШ№вҖҢШӘШұ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұЫҢ ШҜШ§ШұШҜ.

ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҒШұШөШӘ Ъ©ШіШЁ ШҜШұШўЩ…ШҜ Ш§ШұШІЫҢ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ Ш®Ш§ШұШ¬ЫҢ (ЩҶШёЫҢШұ AdMob) ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҶШӘШҙШұ ШҙШҜЩҮ ШӘЩҫШіЩ„ ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ ЩҫЫҢШҙвҖҢШӘШұ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙШҜЩҮ ШЁШ§ШҙЩҶШҜШҢ Ъ©Щ…Ш§Ъ©Ш§ЩҶ ШЁЩҮ Ъ©Ш§Шұ Ш®ЩҲШҜ Ш§ШҜШ§Щ…ЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ЩҲ ШӘШЁЩ„ЫҢШә ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ Ш§Щ…Щ‘Ш§ ШўЩҫШҜЫҢШӘ ЩҶШҙШҜЩҮ ЩҲ ШЁШ§ЪҜвҖҢЩҮШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ.

ШҜШұЫҢШ§ЩҒШӘ SDK ШӘЩҫШіЩ„

ЩҒШ§ЫҢЩ„ zip ШӯШ§ЩҲЫҢ ЩҒШұЫҢЩ…ЩҲШұЪ© ШӘЩҫШіЩ„ ШұШ§ Ш§ШІ ШўШҜШұШі ШІЫҢШұ ШҜШ§ЩҶЩ„ЩҲШҜ Ъ©ЩҶЫҢШҜ. ШҜШұЫҢШ§ЩҒШӘ ЩҒШ§ЫҢЩ„

ЩҒШ§ЫҢЩ„ ЩҒШҙШұШҜЩҮ ШҙШҜЩҮ ШӯШ§ЩҲЫҢ ШҜЩҲ ЩҒШұЫҢЩ…ЩҲШұЪ© Ш§ШіШӘ. ЩҒШұЫҢЩ…ЩҲШұЪ© Universal ЩҮЩ… ШЁШұ ШұЩҲЫҢ ШҙШЁЫҢЩҮвҖҢШіШ§ШІ (Simulator) ЩӮШ§ШЁЩ„ Ш§Ш¬ШұШ§ Ш§ШіШӘ ЩҲ ЩҮЩ… ШЁШұ ШұЩҲЫҢ ШҜШіШӘЪҜШ§ЩҮ ЩҲШ§ЩӮШ№ЫҢ. ЩҒШұЫҢЩ…ЩҲШұЪ© Release ЩҒЩӮШ· ШЁШұ ШұЩҲЫҢ ШҜШіШӘЪҜШ§ЩҮ ЩҲШ§ЩӮШ№ЫҢ ЩӮШ§ШЁЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШіШӘ. ЩҫШі Ш§ШІ ШӘШіШӘ Ъ©ШұШҜЩҶ ШЁШұЩҶШ§Щ…ЩҮ ШЁШ§ ЩҒШұЫҢЩ…ЩҲШұЪ© Universal ШҢ ШЁШұШ§ЫҢ ШұЫҢЩ„ЫҢШІ Ъ©ШұШҜЩҶ ШЁШұЩҶШ§Щ…ЩҮ Ш®ЩҲШҜ ШЁШ§ЫҢШҜ Ш§ШІ ЩҒШұЫҢЩ…ЩҲШұЪ© Release Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШҜЩӮШӘ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ Ш§ШІ ЩҮШұ ЩҶШіШ®ЩҮвҖҢШ§ЫҢ Ъ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ ШҢ ЩҶШ§Щ… ЩҒШ§ЫҢЩ„ ЩҒШұЫҢЩ…ЩҲШұЪ© ШұШ§ ШЁЩҮ TapsellSDKv3.framework ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

Ш§ЩҒШІЩҲШҜЩҶ Framework ШӘЩҫШіЩ„ ШЁЩҮ ЩҫШұЩҲЪҳЩҮ Xcode

ЩҒШ§ЫҢЩ„ ЩҒШұЫҢЩ…ЩҲШұЪ© ШӘЩҫШіЩ„ ШұШ§ ШҜШұ ЩҫЩҲШҙЩҮ ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ Ъ©ЩҫЫҢ Ъ©ЩҶЫҢШҜ ЩҲ ШўЩҶ ШұШ§ ШЁЩҮ ЩҮЩ…ЩҮ Target ЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶШёШұ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ. ШҜШұ ЩӮШіЩ…ШӘ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫШұЩҲЪҳЩҮ ШЁШ®Шҙ General ЩҒШұЫҢЩ…ЩҲШұЪ© ШұШ§ ШҜШұ ШЁШ®Шҙ ЩҮШ§ЫҢ Embedded Binaries ЩҲ Linked Frameworks and Libraries Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ШәЫҢШұЩҒШ№Ш§Щ„ Ъ©ШұШҜЩҶ App Transport Seurity(ATS)

ШЁШ§ Ш§ЩҶШӘШҙШ§Шұ iOS 9 Ш§ЩҫЩ„ ШіЫҢШіШӘЩ… ATS ШұШ§ Щ…Ш№ШұЩҒЫҢ ЩҶЩ…ЩҲШҜ Ъ©ЩҮ Ш·ШЁЩӮ ШўЩҶ Щ„Ш§ШІЩ…ШіШӘ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ Ш§ШІ SSL ШЁШ§ Щ…ШҙШ®ШөШ§ШӘ Ш®Ш§ШөЫҢ ШЁШұШ§ЫҢ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШҙШЁЪ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЩҶШҜ. ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶ШұШҢ Щ„Ш§ШІЩ…ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘШЁЩ„ЫҢШәШ§ШӘ ШӘЩҫШіЩ„ ATS ШҜШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШәЫҢШұЩҒШ№Ш§Щ„ ШҙЩҲШҜ. Ш¬ЩҮШӘ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ Щ…ШҜШ§Ш®Щ„ЩҮ ATS ШҜШұ ЩҫШ®Шҙ ШӘШЁЩ„ЫҢШәШ§ШӘШҢ ЩӮШіЩ…ШӘ ШІЫҢШұ ШұШ§ ШЁЩҮ plist ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.

<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>

Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ URL Schemes

ШЁШ§ Ш§ЩҶШӘШҙШ§Шұ ЩҶШіШ®ЩҮ iOS 9ШҢ ШҙШұЪ©ШӘ Ш§ЩҫЩ„ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШЁЫҢЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ (canOpenUrl) ШұШ§ Щ…ШӯШҜЩҲШҜ Ъ©ШұШҜЩҮ Ш§ШіШӘ ЩҲ ЩҮШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШЁШ§ЫҢШҜ Щ…ШҙШ®Шө Ъ©ЩҶШҜ ЪҶЩҮ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұЫҢ ШұШ§ ШЁШ§ШІ Ш®ЩҲШ§ЩҮШҜ ЩҶЩ…ЩҲШҜ. ШӘЩҫШіЩ„ Ш§ШІ ШӘШ§ШЁШ№ canOpenUrl ШЁШұШ§ЫҢ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢ ШҜШұ Щ…ЩҲШұШҜ Ш§ЩҶШ¬Ш§Щ… Ш№Щ…Щ„ ЩҶЩҮШ§ЫҢЫҢ ШҜШұ ШЁЩҶШұЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШұШ§ЫҢ ЩҒШ№Ш§Щ„ШіШ§ШІЫҢ deep-linking ШЁЫҢЩҶ SDK ШӘЩҫШіЩ„ ЩҲ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒШҢ ШЁШ®Шҙ ШІЫҢШұ ШұШ§ ШЁЩҮ plist ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ.
