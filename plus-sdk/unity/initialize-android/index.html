ШҜШұ ШөЩҲШұШӘ ЩҲШ¬ЩҲШҜ Щ…ШҙЪ©Щ„ ЫҢШ§ Ш§ШЁЩҮШ§Щ… ШЁЩҮ Щ„ЫҢШіШӘ Ш®Ш·Ш§ЩҮШ§ЫҢ ЩҒЩҶЫҢ ШұШ§ЫҢШ¬ Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ ЫҢШ§ ШөЩҒШӯЩҮвҖҢЫҢ Github Issues ШұШ§ ШЁШұШұШіЫҢ ЩҶЩ…Ш§ЫҢЫҢШҜ.

Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ Unity Package ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші

ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ЫІ ШұЩҲШҙ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Ш§ШІ ЩҮШұЪ©ШҜШ§Щ… Ъ©ЩҮ Щ…Ш§ЫҢЩ„ ЩҮШіШӘЫҢШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

Ыұ. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Gradle

Ыұ. Ш§ШЁШӘШҜШ§ unitypackage ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© ШҜШұЫҢШ§ЩҒШӘ ЩҶЩ…Ш§ЫҢЫҢШҜ. ЫІ. Ш§ШІ Ш·ШұЫҢЩӮ Щ…ЩҶЩҲЫҢ ШІЫҢШұ unitypackage ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШұШ§ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ. (Ш§ЪҜШұ ЩҒЩҲЩ„ШҜШұ TapsellPlus Ш§ШІ ЩӮШЁЩ„ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ШҙЩ…Ш§ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Щ„Ш·ЩҒШ§ ШўЩҶ ШұШ§ ЩҫШ§Ъ© Ъ©ЩҶЫҢШҜ)

Assets > Import Package > Custom Package...

Ыі. Ш§ШІ Ш·ШұЫҢЩӮ Щ…ЩҶЩҲЩҮШ§ЫҢ ШІЫҢШұШҢ ШӘЫҢЪ©вҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ Ш§ЫҢШ¬Ш§ШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Custom Gradle ШұШ§ ШЁШІЩҶЫҢШҜ:

Edit > Project Setting... > Player > Publishing Settings > Custom Launcher Gradle Template
Edit > Project Setting... > Player > Publishing Settings > Custom Main Gradle Template
Edit > Project Setting... > Player > Publishing Settings > Custom Base Gradle Template

Ыҙ. ШЁЩҮ ШўШҜШұШі вҖҚвҖҚAssets\Plugins\Android\mainTemplate.gradle ШЁШұЩҲЫҢШҜ ЩҲ :Ш®Ш· Ъ©Ш§Щ…ЩҶШӘ ШҙШҜЩҮвҖҢЫҢ ШЁШ§Щ„Ш§ЫҢ ЩҒШ§ЫҢЩ„ ШұШ§ ШӯШ°ЩҒ ЩҶЩ…Ш§ЫҢЫҢШҜ

// GENERATED BY UNITY. REMOVE THIS COMMENT TO PREVENT OVERWRITING WHEN EXPORTING AGAIN

ШіЩҫШі ШҜШұ ШЁШ®Шҙ dependencies Ш®Ш· ШІЫҢШұ ШұШ§ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ:

dependencies {
    ...
    implementation 'ir.tapsell.plus:tapsell-plus-sdk-unity:2.1.7'
    ...
**DEPS**}

Ш§ЪҜШұ ШӘЪ©ЩҮ Ъ©ШҜ ШІЫҢШұ ЩҶЫҢШІ ШҜШұ ЩҒШ§ЫҢЩ„ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШҙШӘ ШўЩҶ ШұШ§ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ:

android {
  compileOptions {
    sourceCompatibility JavaVersion.VERSION_1_8
    targetCompatibility JavaVersion.VERSION_1_8
  }
}

Ыө. Ш§ЪҜШұ Ш§ШІ ЩҶШіШ®ЩҮ ЫҢЩҲЩҶЫҢШӘЫҢ ЫІЫ°ЫұЫ№ ЩҲ ЩҫШ§ЫҢЫҢЩҶвҖҢШӘШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ Щ„Ш·ЩҒШ§ ШЁЩҮ ШўШҜШұШі Assets\Plugins\Android\baseProjectTemplate.gradle ШЁШұЩҲЫҢШҜ ЩҲ Ш№ШЁШ§ШұШӘ mavenCentral() ШұШ§ ШЁЩҮ ЩҮШұ ШҜЩҲ ШЁШ®Шҙ repositories Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ. ШіЩҫШі ЩҲШұЪҳЩҶ com.android.tools.build:gradle ШұШ§ ШЁЩҮ Ыі.Ы¶.Ы° ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

ШҜЩӮШӘ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ Ъ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҲШұЪҳЩҶ Ыі.Ы¶.Ы° ШЁШұШ§ЫҢ Android Gradle Plugin ШӘЩҶЩҮШ§ ШҜШұ ШөЩҲШұШӘЫҢЪ©ЩҮ Ш§ШІ ЩҶШіШ®ЩҮ ЫҢЩҲЩҶЫҢШӘЫҢ ЫІЫ°ЫұЫ№ ЩҲ ЩҫШ§ЫҢЫҢЩҶвҖҢШӘШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢЪ©ЩҶЫҢШҜ Ш§Щ„ШІШ§Щ…ЫҢ Ш§ШіШӘ. ШІЫҢШұШ§ ШЁШұШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§ЩҶШҜШұЩҲЫҢШҜ ЫҢШ§ШІШҜЩҮ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘЫҢ ШӯШҜШ§ЩӮЩ„ Ш§ШІ Ш§ЫҢЩҶ ЩҲШұЪҳЩҶ ШҜШұ ЩҫШұЩҲЪҳЩҮ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШҙЩҲШҜ. ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЩҶШіШ®ЩҮвҖҢШ§ЫҢ Ш§ШІ Gradle Ъ©ЩҮ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§ЩҶШҜШұЩҲЫҢШҜ ЫҢШ§ШІШҜЩҮ ШұШ§ ШҜШ§ШұШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© Ъ©Щ…Ъ© ШЁЪҜЫҢШұЫҢШҜ.

Ы¶. ШЁШ№ШҜ Ш§ШІ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ЩҲ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ ШҜЫҢЪҜШұ Ш§ШӯШӘЩ…Ш§Щ„Ш§ ШЁЩҮ ШҜЩ„ЫҢЩ„ ШЁШ§Щ„Ш§ ШұЩҒШӘЩҶ ШӯШ¬Щ… Ъ©ШҜ ЩҲ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ЩҲЩӮЩҲШ№ Ш®Ш·Ш§ЫҢ ШІЫҢШұ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘЫҢ MultiDex ШұШ§ ЩҒШ№Ш§Щ„ ЩҶЩ…Ш§ЫҢЫҢШҜ.

D8: Cannot fit requested classes in a single dex file (# methods: 68109 > 65536)

ЩҶШӯЩҲЩҮвҖҢЫҢ ЩҒШ№Ш§Щ„вҖҢШіШ§ШІЫҢ MultiDex ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШўШҜШұШі Assets\Plugins\Android\launcherTemplate.gradle ШЁШұЩҲЫҢШҜ ЩҲ ШӘЪ©ЩҮ Ъ©ШҜ ШІЫҢШұ ШұШ§ ШЁЩҮ ШўЩҶ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ:

android {
  ...
  defaultConfig {
    ...
    multiDexEnabled true
  }
  ...
}

ЫІ. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Resolver

Ыұ. Ш§ШЁШӘШҜШ§ ЩҒШ§ЫҢЩ„ unitypackage resolver ШұШ§ ШҜШ§ЩҶЩ„ЩҲШҜ Ъ©ЩҶЫҢШҜ. ШіЩҫШі Щ…Ш·Ш§ШЁЩӮ ШӘЩҲШ¶ЫҢШӯШ§ШӘ Щ„ЫҢЩҶЪ© ШІЫҢШұ ШўЩҶ ШұШ§ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ:

Щ…ШіШӘЩҶШҜШ§ШӘ ЪҜЩҲЪҜЩ„

ЫІ. Ш§ШІ Ш·ШұЫҢЩӮ Щ…ЩҶЩҲЩҮШ§ЫҢ ШІЫҢШұШҢ ШӘЫҢЪ© Щ…ШұШЁЩҲШ· ШЁЩҮ Jetifier ЩҲ Auto Resolution ШұШ§ ШЁШІЩҶЫҢШҜ. (ШӘЫҢЪ© Щ…ШұШЁЩҲШ· ШЁЩҮ Auto Resolution Ш§Ш¬ШЁШ§ШұЫҢ ЩҶЫҢШіШӘ Ш§Щ…Ш§ ШЁШұШ§ЫҢ ШўЩҶ Ъ©ЩҮ ЩҮШұ ШЁШ§Шұ Щ…Ш¬ШЁЩҲШұ ЩҶШЁШ§ШҙЫҢШҜ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ ШұШ§ ШЁЩҮ ШҙЪ©Щ„ ШҜШіШӘЫҢ Resolve Ъ©ЩҶЫҢШҜ ШЁЩҮШӘШұ Ш§ШіШӘ ШўЩҶ ШұШ§ ЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜ)

Assets > External Dependency Manager > Android Resolver > Settings > Use Jetifier
Assets > External Dependency Manager > Android Resolver > Settings > Enable Auto-Resolution

Ыі. Ш§ШЁШӘШҜШ§ unitypackage ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© ШҜШұЫҢШ§ЩҒШӘ ЩҶЩ…Ш§ЫҢЫҢШҜ.

Ыҙ. Ш§ШІ Ш·ШұЫҢЩӮ Щ…ЩҶЩҲЫҢ ШІЫҢШұ unitypackage ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ШұШ§ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ. (Ш§ЪҜШұ ЩҒЩҲЩ„ШҜШұ TapsellPlus Ш§ШІ ЩӮШЁЩ„ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ШҙЩ…Ш§ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Щ„Ш·ЩҒШ§ ШўЩҶ ШұШ§ ЩҫШ§Ъ© Ъ©ЩҶЫҢШҜ)

Ыө. ШЁШұШ§ЫҢ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ Ш§ШІ Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶ ЩҫЩ„Ш§ЪҜЫҢЩҶ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші Ш§ШІ Ш·ШұЫҢЩӮ Щ…ЩҶЩҲЫҢ ШІЫҢШұ Ш§ЩӮШҜШ§Щ… ШЁЩҮ Resolve Ъ©ШұШҜЩҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ Ъ©ЩҶЫҢШҜ:

ШҜЩӮШӘ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ Ъ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҲШұЪҳЩҶ Ыі.Ы¶.Ы° ШЁШұШ§ЫҢ Android Gradle Plugin Ш§Щ„ШІШ§Щ…ЫҢ Ш§ШіШӘ. ШІЫҢШұШ§ ШЁШұШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§ЩҶШҜШұЩҲЫҢШҜ ЫҢШ§ШІШҜЩҮ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘЫҢ ШӯШҜШ§ЩӮЩ„ Ш§ШІ Ш§ЫҢЩҶ ЩҲШұЪҳЩҶ ШҜШұ ЩҫШұЩҲЪҳЩҮ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШҙЩҲШҜ. ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЩҶШіШ®ЩҮвҖҢШ§ЫҢ Ш§ШІ Gradle Ъ©ЩҮ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§ЩҶШҜШұЩҲЫҢШҜ ЫҢШ§ШІШҜЩҮ ШұШ§ ШҜШ§ШұШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© Ъ©Щ…Ъ© ШЁЪҜЫҢШұЫҢШҜ.

Ы№. ШЁШ№ШҜ Ш§ШІ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ЩҲ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ ШҜЫҢЪҜШұ Ш§ШӯШӘЩ…Ш§Щ„Ш§ ШЁЩҮ ШҜЩ„ЫҢЩ„ ШЁШ§Щ„Ш§ ШұЩҒШӘЩҶ ШӯШ¬Щ… Ъ©ШҜ ЩҲ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ЩҲЩӮЩҲШ№ Ш®Ш·Ш§ЫҢ ШІЫҢШұ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘЫҢ MultiDex ШұШ§ ЩҒШ№Ш§Щ„ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ЩҶШӯЩҲЩҮвҖҢЫҢ ЩҒШ№Ш§Щ„вҖҢШіШ§ШІЫҢ MultiDex ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШўШҜШұШі Assets\Plugins\Android\launcherTemplate.gradle ШЁШұЩҲЫҢШҜ ЩҲ ШӘЪ©ЩҮ Ъ©ШҜ ШІЫҢШұ ШұШ§ ШЁЩҮ ШўЩҶ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ:

Ш§ШЁШӘШҜШ§ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ъ©ШҜЩҮШ§ЫҢ ШӘЩҫШіЩ„ Ш§ШІ ШӘЪ©ЩҮ Ъ©ШҜ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШіЩҫШі ШӘШ§ШЁШ№ ШІЫҢШұ ШұШ§ ШҜШұ ЫҢЪ©ЫҢ Ш§ШІ Ш§ШіЪ©ШұЫҢЩҫШӘвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Ш®ЩҲШҜ Ъ©ЩҮ ШҜШұ Ш§ШЁШӘШҜШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЫҢШҜ.

ШӘЩҶШёЫҢЩ…Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ GDPR

Ш§ШІ ШўЩҶвҖҢШ¬Ш§ Ъ©ЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ЩӮЩҲШ§ЩҶЫҢЩҶ GDPR ШұШ§ ШҜШұ Ш®ШөЩҲШө ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҙШ®ШөЫҢвҖҢШіШ§ШІЫҢ ШҙШҜЩҮ ШұШ№Ш§ЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁЩҮ Ш·ЩҲШұ ЩҫЫҢШҙ ЩҒШұШ¶ Ш§ЪҜШұ Ъ©Ш§ШұШЁШұ ШЁШ§ IP ЫҢЪ©ЫҢ Ш§ШІ Ъ©ШҙЩҲШұЩҮШ§ЫҢ Щ…ШҙЩ…ЩҲЩ„ Ш§ЫҢЩҶ ЩӮШ§ЩҶЩҲЩҶ Ш§ШІ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШҙЩ…Ш§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶШҜШҢ ШҜЫҢШ§Щ„ЩҲЪҜЫҢ ШҜШұ Ш§ЫҢЩҶ Ш®ШөЩҲШө ШЁЩҮ Ъ©Ш§ШұШЁШұ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЪҜШұ ШӘЩ…Ш§ЫҢЩ„ ШҜШ§ШұЫҢШҜ ШӘШ§ ШЁЩҮ Ш¬Ш§ЫҢ ШӘШөЩ…ЫҢЩ… Ъ©Ш§ШұШЁШұШҢ Ш®ЩҲШҜШӘШ§ЩҶ ШҜШіШӘШұШіЫҢ Щ„Ш§ШІЩ… ШұШ§ ШӘШ№ЫҢЫҢЩҶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШӘЪ©ЩҮ Ъ©ШҜ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ. ШӘЩҲШ¬ЩҮ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ Ъ©ЩҮ Ш§ЫҢЩҶ ШӘЪ©ЩҮ Ъ©ШҜ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘЫҢ ЩҫШі Ш§ШІ Initialize ШҙШҜЩҶ ШӘЩҫШіЩ„ ЩҫЩ„Ш§Ші ЩҲ ЩҫЫҢШҙ Ш§ШІ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ШөШҜШ§ ШІШҜЩҮ ШҙЩҲШҜ ШӘШ§ ЩҶШӘЫҢШ¬ЩҮвҖҢЫҢ ШўЩҶ ШҜШұ ШҜШұШ®ЩҲШ§ШіШӘ ШҙЩ…Ш§ Ш§Ш№Щ…Ш§Щ„ ШҙШҜЩҮ ШЁШ§ШҙШҜ.
