  • ШҜШұЫҢШ§ЩҒШӘ ЩҶШӘЫҢШ¬ЩҮ ШӘШЁЩ„ЫҢШә Ш¬Ш§ЫҢШІЩҮвҖҢШ§ЫҢ

вҡ  ЩҮШҙШҜШ§Шұ! Ш§ЫҢЩҶ SDK ШҜЫҢЪҜШұ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ вҡ 

Ш§ЪҜШұ ШӘШ§Ъ©ЩҶЩҲЩҶ ШЁШұШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҜШұЩҲЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш§ШІ ШӘЩҫШіЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ШұШҜЫҢШҜШҢ ШЁЩҮШӘШұ Ш§ШіШӘ ШІЫҢЩҶвҖҢЩҫШі Ш§ШІ ШӘЩҫШіЩ„вҖҢЩҫЩ„Ш§Ші Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ШӘЩҫШіЩ„ ЩҫЩ„Ш§ШіШҢ Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШҜШ§ШұШ§ ШЁЩҲШҜЩҶ ШӘЩ…Ш§Щ… Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ ШӘЩҫШіЩ„ШҢ Ш§Щ„ЪҜЩҲШұЫҢШӘЩ…вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙЩ…ЩҶШҜШ§ЩҶЩҮвҖҢШӘШұШҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Щ…ШӘЩҶЩҲШ№вҖҢШӘШұ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұЫҢ ШҜШ§ШұШҜ.

ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҒШұШөШӘ Ъ©ШіШЁ ШҜШұШўЩ…ШҜ Ш§ШұШІЫҢ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ Ш®Ш§ШұШ¬ЫҢ (ЩҶШёЫҢШұ AdMob) ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҶШӘШҙШұ ШҙШҜЩҮ ШӘЩҫШіЩ„ ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ ЩҫЫҢШҙвҖҢШӘШұ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙШҜЩҮ ШЁШ§ШҙЩҶШҜШҢ Ъ©Щ…Ш§Ъ©Ш§ЩҶ ШЁЩҮ Ъ©Ш§Шұ Ш®ЩҲШҜ Ш§ШҜШ§Щ…ЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ЩҲ ШӘШЁЩ„ЫҢШә ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ Ш§Щ…Щ‘Ш§ ШўЩҫШҜЫҢШӘ ЩҶШҙШҜЩҮ ЩҲ ШЁШ§ЪҜвҖҢЩҮШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ.

ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Ш¬Ш§ЫҢШІЩҮвҖҢШ§ЫҢ ЩҲ ШўЩҶЫҢ (ЩҮЩ… ЩҲЫҢШҜЫҢЩҲвҖҢ ЩҲ ЩҮЩ… ШЁЩҶШұЫҢ) ШЁЩҮ ЫҢЪ© ШөЩҲШұШӘ Ш§ШіШӘ. ЩҒЩӮШ· Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ ЩҶЩҲШ№ ШӘШЁЩ„ЫҢШәЪҜШ§ЩҮ ШұШ§ Ш§ШІ ЩҫЩҶЩ„ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ.

ШіШ§Ш®ШӘ ШӘШЁЩ„ЫҢШәЪҜШ§ЩҮ

Ш§ШЁШӘШҜШ§ Ш§ШІ ЩҫЩҶЩ„ ШӘЩҫШіЩ„ ЫҢЪ© ШӘШЁЩ„ЫҢШәвҖҢЪҜШ§ЩҮ Ш§ШІ ЩҶЩҲШ№ЫҢ Ъ©ЩҮ Щ…Ш§ЫҢЩ„ ЩҮШіШӘЫҢШҜ ШЁШіШ§ШІЫҢШҜ.

ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә

ШЁШ§ Ш§Ш¬ШұШ§ЫҢ Ъ©ШҜ ШІЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұШ®ЩҲШ§ШіШӘ ЫҢЪ© ШӘШЁЩ„ЫҢШә ШЁШҜЩҮЫҢШҜ.

Tapsell.requestAd(CONTEXT,
        ZONE_ID,
        new TapsellAdRequestOptions(),
        new TapsellAdRequestListener() {
            @Override
            public void onAdAvailable(String adId) {
            }

            @Override
            public void onError(String message) {
            }
        });

ЩҲШұЩҲШҜЫҢ Ш§ЩҲЩ„ CONTEXT Щ…ЫҢШЁШ§ШҙШҜ.
ЩҲШұЩҲШҜЫҢ ШҜЩҲЩ… ZONE_ID ШЁШұШ§ШЁШұ ШЁШ§ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШәвҖҢЪҜШ§ЩҮЫҢ ЩҮШіШӘ Ъ©ЩҮ ШҜШұ ЩҫЩҶЩ„ ШіШ§Ш®ШӘЩҮвҖҢШ§ЫҢШҜ.
ЩҲШұЩҲШҜЫҢ ШіЩҲЩ… OPTIONS Ш§ШІ ЩҶЩҲШ№ TapsellAdRequestOptions Ш§ШіШӘ. ШӘЩҶШёЫҢЩ…Ш§ШӘ ШІЩ…Ш§ЩҶ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Ш·ШұЫҢЩӮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЩҶШ¬Ш§Щ… ШЁШҜЩҮЫҢШҜ.
ЩҲШұЩҲШҜЫҢ ЪҶЩҮШ§ШұЩ… Ш§ШІ ЩҶЩҲШ№ TapsellAdRequestListener Ш§ШіШӘ.

Щ…ШӘШҜЩҮШ§ЫҢ TapsellAdRequestListener Щ…Ш·Ш§ШЁЩӮ Ш¬ШҜЩҲЩ„ ШІЫҢШұ Ш§ШіШӘ ЩҲ ЩҶШӘЫҢШ¬ЩҮ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ШЁШ§ Ъ©Щ…Ъ© Ш§ЫҢЩҶ ШЁШ®Шҙ ШЁШұЩ…ЫҢЪҜШұШҜШҜ.

Ш№Щ…Щ„Ъ©ШұШҜЩ…ШӘШҜ
ШӘШЁЩ„ЫҢШ№ ШўЩ…Ш§ШҜЩҮ ЩҶЩ…Ш§ЫҢШҙ Ш§ШіШӘ ЩҲ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә ШҜШұ Щ…ШӘШәЫҢЫҢШұ adId ЩӮШұШ§Шұ ШҜШ§ШұШҜ.onAdAvailable
Ш®Ш·Ш§ЫҢЫҢ ШұШ® ШҜШ§ШҜЩҮ Ш§ШІ Ш·ШұЫҢЩӮ message Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш®Ш·Ш§ ШұШ§ ШЁШЁЫҢЩҶЫҢШҜ.onError

Ш§ЪҜШұ ШӘЩ…Ш§ЫҢЩ„ ШҜШ§ШұЫҢШҜ ШҜШұ Ъ©Ш§Щ„ШЁЪ© onError Щ…Ш¬ШҜШҜШ§ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә Ъ©ЩҶЫҢШҜШҢ ШӯШӘЩ…Ш§ Ш§ЫҢЩҶ Ъ©Ш§Шұ ШұШ§ ШЁЩҮ Ъ©Щ…Ъ© Щ…ШӘШәЫҢШұЫҢ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШҙЩ…Ш§ШұЩҶШҜЩҮ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЫҢШҜ. ШІЫҢШұШ§ ШЁЩҮ Ъ©Щ…Ъ© ШўЩҶ Щ…ШӘШәЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ШӯШҜЩҲШҜЫҢШӘ ШӘШ№ШҜШ§ШҜ ШҜЩҒШ№Ш§ШӘ ШұШ§ ШЁШұШ§ЫҢ ШҜШұШ®ЩҲШ§ШіШӘ Щ„ШӯШ§Шё Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ ЩҲЩӮШӘЫҢ Ш§ЫҢЩҶ Ш¬Ш§ЫҢЪҜШ§ЩҮ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ ШұШ§ Ш§ШІ ЩҫЩҶЩ„ ШәЫҢШұЩҒШ№Ш§Щ„ ЩҶЩ…ЩҲШҜЫҢШҜШҢ Ш§ЪҜШұ ШЁШҜЩҲЩҶ Щ…ШӯШҜЩҲШҜ Ъ©ШұШҜЩҶ ШҜЩҒШ№Ш§ШӘШҢ ЩҮШұ ШЁШ§Шұ ШҜШұ Ъ©Ш§Щ„ШЁЪ© onError Щ…Ш¬ШҜШҜШ§ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ШҜЩҮЫҢШҜШҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢШӘШ§ЩҶ ШҜШұ ЫҢЪ© ШӯЩ„ЩӮЩҮвҖҢЫҢ ШЁЫҢвҖҢЩҶЩҮШ§ЫҢШӘ Щ…ЫҢвҖҢШ§ЩҒШӘШҜ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШўЩҶ Щ…Ш®ШӘЩ„ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜШұШ®ЩҲШ§ШіШӘ

ШЁШ§ ШұЩҲШҙ ШІЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҶЩҲШ№ cache ШҙШҜЩҶ ЩҲЫҢШҜЫҢЩҲ ШұШ§ ШЁШ§ Ъ©Щ…Ъ© ЩҲШұЩҲШҜЫҢ options ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

TapsellAdRequestOptions options = new TapsellAdRequestOptions();
options.setCacheType(CACHE_TYPE);

Щ…ЩӮШҜШ§Шұ CACHE_TYPE Щ…ЫҢШӘЩҲШ§ЩҶШҜ ШЁШұШ§ШЁШұ Щ…ЩӮШ§ШҜЫҢШұ ШІЫҢШұ ШЁШ§ШҙШҜ.

ШӘЩҲШ¶ЫҢШӯШ§ШӘЩ…ЩӮШҜШ§Шұ
ЩҮЩҶЪҜШ§Щ… ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШҢ ШҙШұЩҲШ№ ШЁЩҮ ШҜШ§ЩҶЩ„ЩҲШҜ ЩҲЫҢШҜЫҢЩҲ Щ…ЫҢЪ©ЩҶШҜTapsellAdRequestOptions.CACHE_TYPE_STREAMED
ЩӮШЁЩ„ Ш§ШІ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШҢ ЩҲЫҢШҜЫҢЩҲ ШұШ§ ШҜШ§ЩҶЩ„ЩҲШҜ Щ…ЫҢЪ©ЩҶШҜTapsellAdRequestOptions.CACHE_TYPE_CACHED

ШӘЩҶЩҮШ§ ШІЩ…Ш§ЩҶЫҢ Ш§ШІ CACHE_TYPE_CACHED Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ Ш§ШӯШӘЩ…Ш§Щ„ ШҜЫҢШҜЩҶ ЩҲЫҢШҜЫҢЩҲ ШӘЩҲШіШ· Ъ©Ш§ШұШЁШұ ШІЫҢШ§ШҜ ШЁШ§ШҙШҜШҢ ШӘШ§ Щ…ШөШұЩҒ Ш§ЫҢЩҶШӘШұЩҶШӘ ШЁШ§Щ„Ш§ ЩҶШұЩҲШҜ.

ЩҲШұЩҲШҜЫҢ Ш§ЩҲЩ„ CONTEXT Щ…ЫҢШЁШ§ШҙШҜ.
ЩҲШұЩҲШҜЫҢ ШҜЩҲЩ… ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШәвҖҢЪҜШ§ЩҮ Ш§ШіШӘ. ЩҲШұЩҲШҜЫҢ ШіЩҲЩ… ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШә Ъ©ЩҮ ШҜШұ ШІЩ…Ш§ЩҶ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ШӘЩҲШіШ· Щ…ШӘШҜ onAdAvailable ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ЩҲШұЩҲШҜЫҢ ЪҶЩҮШ§ШұЩ… Ш§ШІ ЩҶЩҲШ№ TapsellShowOptions Ш§ШіШӘ. ШӘЩҶШёЫҢЩ…Ш§ШӘ ШІЩ…Ш§ЩҶ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Ш·ШұЫҢЩӮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЩҶШ¬Ш§Щ… ШЁШҜЩҮЫҢШҜ.
ЩҲШұЩҲШҜЫҢ ЩҫЩҶШ¬Щ… Ш§ШІ ЩҶЩҲШ№ TapsellAdShowListener Ш§ШіШӘ. Щ…ШұШ§ШӯЩ„ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә ШұШ§ Ш§ШІ Ш§ЫҢЩҶ Ш·ШұЫҢЩӮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.

Щ…ШӘШҜЩҮШ§ЫҢ TapsellAdShowListener Щ…Ш·Ш§ШЁЩӮ Ш¬ШҜЩҲЩ„ ШІЫҢШұ Ш§ШіШӘ.

Ш№Щ…Щ„Ъ©ШұШҜЩ…ШӘШҜ
ЩҮЩҶЪҜШ§Щ… ШЁШ§ШІ ШҙШҜЩҶ ШӘШЁЩ„ЫҢШә ШөШҜШ§ ШІШҜЩҮ Щ…ЫҢШҙЩҲШҜ.onOpened
ЩҮЩҶЪҜШ§Щ… ШЁШіШӘЩҮ ШҙШҜЩҶ ШӘШЁЩ„ЫҢШә ШөШҜШ§ ШІШҜЩҮ Щ…ЫҢШҙЩҲШҜ.onClosed
ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ Ш®Ш·Ш§ЫҢЫҢ ШҜШұ ЩҫШұЩҲШіЩҮ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә ЩҫЫҢШҙ ШЁЫҢШ§ЫҢШҜ ШөШҜШ§ ШІШҜЩҮ Щ…ЫҢШҙЩҲШҜ.onError
ЩҲШ¶Ш№ЫҢШӘ ШҜШұЫҢШ§ЩҒШӘ Ш¬Ш§ЫҢШІЩҮ ШҜШұ ШӘШЁЩ„ЫҢШәШ§ШӘ Ш¬Ш§ЫҢШІЩҮвҖҢШ§ЫҢ ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢШҜЩҮШҜ.onRewarded

ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҶЩ…Ш§ЫҢШҙ

ШӘЩҶШёЫҢЩ…Ш§ШӘ ШІЩ…Ш§ЩҶ ЩҶЩ…Ш§ЫҢШҙ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…Ш·Ш§ШЁЩӮ ШұЩҲШҙ ШІЫҢШұ ШЁШ§ Ъ©Щ…Ъ© ЩҲШұЩҲШҜЫҢ showOptions ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

ШЁЩҮ ЩҮШұ ЫҢЪ© Ш§ШІ Щ…ШӘШҜЩҮШ§ЫҢ ШЁШ§Щ„Ш§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ЩӮШҜШ§Шұ true ЫҢШ§ false ШұШ§ ШЁШҜЩҮЫҢШҜ ШӘШ§ ШўЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҒШ№Ш§Щ„ ЫҢШ§ ШәЫҢШұ ЩҒШ№Ш§Щ„ ШЁШҙЩҲШҜ. Щ…ЩӮШ§ШҜЫҢШұ ЩӮШ§ШЁЩ„ Ш§ЩҶШӘШ®Ш§ШЁ ШЁШұШ§ЫҢ setRotationMode ЩҫШ§ЫҢЫҢЩҶШӘШұ ШӘЩҲШ¶ЫҢШӯ ШҜШ§ШҜЩҮ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

Ъ©Ш§ШұШЁШұШҜ ЩҮШұ ЫҢЪ© Ш§ШІ Ш§ЫҢЩҶ Щ…ШӘШҜЩҮШ§ Щ…Ш·Ш§ШЁЩӮ Ш¬ШҜЩҲЩ„ ШІЫҢШұ Ш§ШіШӘ.

Ш№Щ…Щ„Ъ©ШұШҜЩ…ШӘШҜ
ШәЫҢШұ ЩҒШ№Ш§Щ„ Ъ©ШұШҜЩҶ ШҜЪ©Щ…ЩҮ ШЁШ§ШІЪҜШҙШӘ ШҜШұ ЩҮЩҶЪҜШ§Щ… ЩҶЩ…Ш§ЫҢШҙ.setBackDisabled
ЩҒШ№Ш§Щ„ Ъ©ШұШҜЩҶ ШӯШ§Щ„ШӘ Immersive ЩҮЩҶЪҜШ§Щ… ЩҶЩ…Ш§ЫҢШҙ.setImmersiveMode

ШӘШ№ЫҢЫҢЩҶ Ш¬ЩҮШӘ ЪҜЩҲШҙЫҢ ЩҮЩҶЪҜШ§Щ… ЩҶЩ…Ш§ЫҢШҙ.setRotationMode

Щ…ЩӮШ§ШҜЫҢШұЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ setRotationMode ШЁШҜЩҮЫҢШҜ Щ…Ш·Ш§ШЁЩӮ Ш¬ШҜЩҲЩ„ ШІЫҢШұ Ш§ШіШӘ.
