ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ЩҮЩ…ШіШ§ЩҶ ШҜШұ React Native

вҡ  ЩҮШҙШҜШ§Шұ! Ш§ЫҢЩҶ SDK ШҜЫҢЪҜШұ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ вҡ 

Ш§ЪҜШұ ШӘШ§Ъ©ЩҶЩҲЩҶ ШЁШұШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҜШұЩҲЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш§ШІ ШӘЩҫШіЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ШұШҜЫҢШҜШҢ ШЁЩҮШӘШұ Ш§ШіШӘ ШІЫҢЩҶвҖҢЩҫШі Ш§ШІ ШӘЩҫШіЩ„вҖҢЩҫЩ„Ш§Ші Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ШӘЩҫШіЩ„ ЩҫЩ„Ш§ШіШҢ Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШҜШ§ШұШ§ ШЁЩҲШҜЩҶ ШӘЩ…Ш§Щ… Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ ШӘЩҫШіЩ„ШҢ Ш§Щ„ЪҜЩҲШұЫҢШӘЩ…вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙЩ…ЩҶШҜШ§ЩҶЩҮвҖҢШӘШұШҢ ШӘШЁЩ„ЫҢШәШ§ШӘ Щ…ШӘЩҶЩҲШ№вҖҢШӘШұ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұЫҢ ШҜШ§ШұШҜ.

ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҒШұШөШӘ Ъ©ШіШЁ ШҜШұШўЩ…ШҜ Ш§ШұШІЫҢ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШ§ШӘ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ ШӘШЁЩ„ЫҢШәШ§ШӘЫҢ Ш®Ш§ШұШ¬ЫҢ (ЩҶШёЫҢШұ AdMob) ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҶШӘШҙШұ ШҙШҜЩҮ ШӘЩҫШіЩ„ ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ ЩҫЫҢШҙвҖҢШӘШұ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙШҜЩҮ ШЁШ§ШҙЩҶШҜШҢ Ъ©Щ…Ш§Ъ©Ш§ЩҶ ШЁЩҮ Ъ©Ш§Шұ Ш®ЩҲШҜ Ш§ШҜШ§Щ…ЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ЩҲ ШӘШЁЩ„ЫҢШә ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ Ш§Щ…Щ‘Ш§ ШўЩҫШҜЫҢШӘ ЩҶШҙШҜЩҮ ЩҲ ШЁШ§ЪҜвҖҢЩҮШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ.

ШіШ§Ш®ШӘ ШӘШЁЩ„ЫҢШәЪҜШ§ЩҮ

Ш§ШЁШӘШҜШ§ Ш§ШІ ЩҫЩҶЩ„ ШӘЩҫШіЩ„ ЫҢЪ© ШӘШЁЩ„ЫҢШәЪҜШ§ЩҮ (zone) ЩҮЩ…ШіШ§ЩҶ ШЁШіШ§ШІЫҢШҜ ЩҲ zoneId ШұШ§ ШІЩ…Ш§ЩҶ ШҜШұШ®ЩҲШ§ШіШӘ ЩҲ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә

Ш§ШЁШӘШҜШ§ Щ…Ш§ЪҳЩҲЩ„ ШӘЩҫШіЩ„ ЩҲ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ AdVideo ШұШ§ ШЁЩҮ Ъ©ШҜ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ:

import Tapsell, { AdVideo } from 'react-native-tapsell'

Ш¬ЩҮШӘ Ш§ШұШіШ§Щ„ ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә ЩҮЩ…ШіШ§ЩҶШҢ Ш§ШІ ШӘШ§ШЁШ№ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

Tapsell.requestNativeЩ°VideoAd(zoneId, onAdAvailable, onNoAdAvailable, OnNoNetwork, onError);

ЩҲШұЩҲШҜЫҢ zoneIdШҢ ШҙЩҶШ§ШіЩҮ ШӘШЁЩ„ЫҢШәвҖҢЪҜШ§ЩҮ Ш§ШіШӘ Ъ©ЩҮ ШЁШ§ЫҢШҜ ШўЩҶ ШұШ§ Ш§ШІ ШҜШ§ШҙШЁЩҲШұШҜ ШӘЩҫШіЩ„ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.

Ш§Ъ©ШҙЩҶвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҲ ШҙШұШ§ЫҢШ· Ш§Ш¬ШұШ§ ШҙШҜЩҶ ШўЩҶвҖҢЩҮШ§ ШҜШұ Ш¬ШҜЩҲЩ„ ШІЫҢШұ ШўЩ…ШҜЩҮ Ш§ШіШӘ:

ШӘЩҲШ¶ЫҢШӯШ§ШӘШӘШ§ШЁШ№
ЩҮЩҶЪҜШ§Щ…ЫҢ Ъ©ЩҮ ЩҮШұ ЩҶЩҲШ№ Ш®Ш·Ш§ЫҢЫҢ ШҜШұ ЩҫШұЩҲШіЩҮвҖҢЫҢ ШҜШұЫҢШ§ЩҒШӘ ШӘШЁЩ„ЫҢШә ШЁЩҲШ¬ЩҲШҜ ШЁЫҢШ§ЫҢШҜonError(zoneId : string, error : string)
ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ ШӘШЁЩ„ЫҢШә ШҜШұЫҢШ§ЩҒШӘ ШҙШҜЩҮ ЩҲ ШўЩ…Ш§ШҜЩҮвҖҢЫҢ ЩҶЩ…Ш§ЫҢШҙ ШЁШ§ШҙШҜonAdAvailable(zoneId : string, adId : string)
ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ ШӘШЁЩ„ЫҢШәЫҢ ШЁШұШ§ЫҢ ЩҶЩ…Ш§ЫҢШҙ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜonNoAdAvailable(zoneId : string)
ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШҙШЁЪ©ЩҮ Щ…ЩҲШ¬ЩҲШҜ ЩҶШЁШ§ШҙШҜonNoNetwork(zoneId : string)

ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШә

ШЁШұШ§ЫҢ ЩҶЩ…Ш§ЫҢШҙ ШӘШЁЩ„ЫҢШәШҢ Щ…ЫҢвҖҢШЁШ§ЫҢШіШӘ Ш§ШІ ЩҒЫҢЩ„ШҜЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ ШҜШұ ШўШЁШ¬Ъ©ШӘ adData ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ Ш§ШІ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ AdVideo Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ЩҶЩ…ЩҲЩҶЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ AdVideo ШҜШұ Ъ©ШҜ ШІЫҢШұ ШўЩ…ШҜЩҮ Ш§ШіШӘ.
