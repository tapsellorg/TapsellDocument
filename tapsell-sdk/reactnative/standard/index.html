ШіШ§Ш®ШӘ ШӘШЁЩ„ЫҢШәЪҜШ§ЩҮ

Ш§ШЁШӘШҜШ§ Ш§ШІ ЩҫЩҶЩ„ ШӘЩҫШіЩ„ ЫҢЪ© ШӘШЁЩ„ЫҢШәвҖҢЪҜШ§ЩҮ Ш§ШІ ЩҶЩҲШ№ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШЁШіШ§ШІЫҢШҜ.

ШҜШұШ®ЩҲШ§ШіШӘ ШӘШЁЩ„ЫҢШә

Ш§ШЁШӘШҜШ§ Щ…Ш§ЪҳЩҲЩ„ ШӘЩҫШіЩ„ ЩҲ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ BannerAd ШұШ§ ШЁЩҮ Ъ©ШҜ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢШҜ:

import Tapsell, { BannerAd } from 'react-native-tapsell'

Ш¬ЩҮШӘ ЩҶЩ…Ш§ЫҢШҙ ШЁЩҶШұ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜШҢ ШЁШ§ЫҢШҜ Щ…ШӯЩ„ЫҢ ШЁШұШ§ЫҢ ЩҶЩ…Ш§ЫҢШҙ ШўЩҶ ШҜШұ ШөЩҒШӯЩҮ ШҜШұ ЩҶШёШұ ШЁЪҜЫҢШұЫҢШҜ. ШЁЩҶШұ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜШҢ ШҜШ§ШұШ§ЫҢ ШіШ§ЫҢШІЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ SDK ШӘЩҫШіЩ„ Щ…ШҙШ®Шө ШҙШҜЩҮвҖҢШ§ЩҶШҜ. Ш¬ЩҮШӘ ЩҶЩ…Ш§ЫҢШҙ ШЁЩҶШұШҢ Ш§ШІ component ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:

<BannerAd
  zoneId={STANDARD_BANNER_ZONE_ID}
  bannerType={Tapsell.BANNER_300x250}
  />

Щ…ЩӮШҜШ§Шұ zoneId Ъ©Щ„ЫҢШҜЫҢ ШіШӘ Ъ©ЩҮ ШЁШ№ШҜ Ш§ШІ ШіШ§Ш®ШӘ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШҜШұ ЩҫЩҶЩ„ ЩҲ Ш«ШЁШӘ ЫҢЪ© zone Ш§ШІ ЩҶЩҲШ№ ШЁЩҶШұЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢЪ©ЩҶЫҢШҜ. ШЁШ§ ШӘШәЫҢЫҢШұ ЩҒЫҢЩ„ШҜ zoneId ШӘШЁЩ„ЫҢШә ШЁШ§ШұЪҜШІШ§ШұЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ.

ЩҲШұЩҲШҜЫҢ BannerType Ш§ЩҶШҜШ§ШІЩҮвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ШұШ§ ШЁЫҢШ§ЩҶ Щ…ЫҢЪ©ЩҶШҜ ЩҲ ШҙШ§Щ…Щ„ Щ…ЩӮШ§ШҜЫҢШұ ШІЫҢШұ Щ…ЫҢвҖҢШЁШ§ШҙШҜ:
